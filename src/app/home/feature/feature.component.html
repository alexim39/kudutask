Get Task Notification

Get on the go notification on new task, task progress and completed status.

KuduTask will help you not to miss task deadline.

Assign Task To Members

Assign task to one or more team member, monitor task progress in real-time.

If need be, you can add members, remove members or re-assign task to different persons.

You can mark a task as private (personal Todo) or for team

Share Task With Team

You can decide to share a task with your team members.

This enables every team member get abreast task progress.

Share it only if you like and keep it private if otherwise.
